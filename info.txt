Correct Process of Creating and managing repository from terminal:

mkdir folder-name
cd folder-name

touch file1 file2 file3 ...

git init
git add .
git commit -m "Your message"
git branch -M main
gh repo create practice-repo2 --public --source=. --remote=origin --push

Normal Workflow (daily usage):
Check repo status:
git status

Stage changes:
git add .

Commit:
git commit -m "Your message"
git branch -M main

Push to GitHub:
git push

Pull latest changes
git pull
